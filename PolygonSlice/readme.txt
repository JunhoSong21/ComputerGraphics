화면의 좌 우측 가장자리 임의의 위치에서 도형이 화면의 반대편으로 날아온다. 구현 O
날아오는 경로 : 사선 ( 화면의 40%를 지나면 낙하 속도 증가)
도형의 종류 : 삼각형, 사각형

아래에는 바구니가 좌우로 움직이고 있다. 구현 O
마우스를 클릭, 드래그하여 도형을 슬라이스 한다. 구현 O
도형이 잘려서 작은 조각이 되어 아래로 떨어진다. 구현 O (정해진 모양대로 잘리게 구현)
조각이 떨어질 때는 중력이 적용되어 떨어진다. (좌우 운동 없이 수직낙하하도록 구현)
떨어진 조각이 바구니에 닿으면 바구니 위에 놓여져 있다. (특정 위치에 고정되게 구현)
슬라이스 하지 못하면 반대편으로 사라진다. 구현 O
종료 명령어가 입력될 때까지 계속 도형이 날아온다. 구현 O

키보드 명령어
도형의 모드 : LINE/FILL (m키 입력으로 변경)
날아오는 속도 변경하기 : 0, 1, 2, 3단계. (기본 1단계. [ 키 속도 증가 ] 키 속도 감소)
프로그램 종료 : q
